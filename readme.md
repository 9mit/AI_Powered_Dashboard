India Development Goals Dashboard
This is an AI-powered analytics dashboard built with Streamlit, designed to monitor and visualize India's progress towards key development goals across education, healthcare, infrastructure, and digital/financial inclusion. It features interactive data visualizations, simulated AI/ML insights, and integrates with the Google Gemini API for generating intelligent summaries, answering data-related questions, and providing policy recommendations.

Features
Interactive Filters: Select specific states and districts across India to view localized data.

Simulated AI/ML Insights: Displays latest values, year-over-year changes, forecasts for the next year, and anomaly detection for key indicators.

Time-Series Visualizations: Interactive line charts powered by Plotly, showing trends for each development indicator over a 10-year period (2015-2024).

AI-Powered Policy Summary: Generates concise, actionable summaries of the displayed data for policymakers, NGOs, and citizens using the Gemini API.

Ask a Question about the Data: Allows users to ask natural language questions about the visualized data, with answers provided by the API.

Policy Recommendation Generator: Provides actionable policy recommendations based on the analyzed data and trends, generated by the API.

Setup Instructions
Follow these steps to set up and run the application on your local machine.

Prerequisites
Python 3.8+ installed on your system.

1. Create a Virtual Environment (Recommended)
It's best practice to use a virtual environment to manage project dependencies.

# Navigate to your project directory
cd india_dev_dashboard_streamlit

# Create a virtual environment
python -m venv venv

# Activate the virtual environment
# On Windows:
.\venv\Scripts\activate
# On macOS/Linux:
source venv/bin/activate

2. Install Dependencies
Once your virtual environment is active, install the required Python packages:

pip install -r requirements.txt

3. Set up your Google Gemini API Key
The application uses the Google Gemini API for its AI-powered features. You need to obtain an API key from Google AI Studio.

Go to Google AI Studio.

Create a new API key.

In the root directory of your project (india_dev_dashboard_streamlit/), create a new file named .env.

Add your API key to this file in the following format:

GEMINI_API_KEY=YOUR_GEMINI_API_KEY_HERE

Replace YOUR_GEMINI_API_KEY_HERE with the actual API key you obtained.
Important: Do not commit your .env file to version control (e.g., Git repositories).

4. Run the Application
With the virtual environment active and dependencies installed, you can run the Streamlit application:

streamlit run app.py

This command will open the dashboard in your default web browser. If it doesn't open automatically, look for a URL (usually http://localhost:8501) in your terminal.

Usage
Use the "Select State" and "Select District" dropdowns to filter the data.

Review the "AI/ML Insights" panel for quick summaries of current status, forecasts, and anomalies.

Explore the "Time-Series Visualizations" to see historical trends.

Use the "AI-Powered Policy Summary", "Ask a Question about the Data", and "Policy Recommendation Generator" sections to get AI-driven insights and recommendations (requires a valid GEMINI_API_KEY).

Data Source
The data used in this dashboard is simulated for demonstration purposes. In a real-world scenario, this data would be pulled from official government portals like data.gov.in or NITI Aayog's NDAP.
